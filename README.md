# Loan Default Prediction

## Problem Statement
Predict whether a loan applicant will default on their 
Home Credit loan based on 122 financial and personal attributes.

## Dataset
Home Credit Default Risk — Kaggle Competition
- 307,511 loan applications
- 122 features
- Target: 1 = defaulted, 0 = repaid

## Source
https://www.kaggle.com/competitions/home-credit-default-risk

## Pipeline
1. Data Loading & Setup
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Engineering
5. Class Imbalance Handling (SMOTE)
6. Model Training & Comparison
7. Hyperparameter Tuning
8. Threshold Optimization
9. SHAP Explainability
10. Streamlit Deployment

## Tech Stack
Python, Pandas, Scikit-learn, XGBoost, SHAP, Streamlit
